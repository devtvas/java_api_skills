> java_api_skills
api contruida em java
para mostrar conhecimentos basicos

##como baixar o projeto
- clone o projeto
- use uma ide (netbens ou intellij)
- instale maven e o spring
- selecione o arq. PadwanTestApplication e rode o projeto
